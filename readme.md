# Oblique Strategies JSON API

This is a JSON API to deliver Brian Eno and Peter Schmidt's Oblique Strategies.

## Example queries

*	/all

*	/draw

*	/edition/3/all

*	/edition/3/draw

*	/edition/3/cardnumber/5

*	/id/9
